# Lokale Vacaturezoek-Widget voor Websites

Lichtgewicht insluitbare vacaturezoek-widget voor websites, blogs, directories en mediaplatforms.

Voeg een volledig responsieve lokale vacaturezoekervaring toe aan uw website met één enkele iframe-embed. De widget bevat zoeken op trefwoord, locatie-automatisch aanvullen, paginering en automatische iframe-hoogte aanpassing.

Perfect voor:

* Vacaturesites
* Blogs
* Lokale nieuwswebsites
* Vastgoedportalen
* Carrièrepagina's
* SEO-landingspagina's
* Publicatieplatforms

---

# Functies

* Responsief ontwerp
* Automatisch aanvullen van trefwoorden
* Automatisch aanvullen van locaties
* Dynamische automatische iframe-hoogte
* Pagineringondersteuning
* Mobielvriendelijk
* Snel lichtgewicht insluiten
* Eenvoudige installatie
* Domeinoverschrijdend compatibel
* SEO-vriendelijke integratie

# Installatie

Kopieer en plak deze code in uw website:

```html
<div class="jobs-widget-wrap"><iframe id="jobsWidget" src="https://jobs-in.us/display/jobs-search-widget.php" loading="lazy"></iframe><div class="jobs-widget-credit">Lokale vacaturezoek-widget door <a href="https://jobs-in.us/find/country,US" target="_blank">jobs-in.us</a></div></div><style>.jobs-widget-wrap{margin:40px 0}#jobsWidget{width:100%;height:760px;border:0;overflow:hidden;border-radius:24px;background:#f5f5f5;transition:height .25s ease}.jobs-widget-credit{margin-top:10px;text-align:right;font-size:13px;color:#777}.jobs-widget-credit a{color:#1a73e8;text-decoration:none}</style><script>window.addEventListener('message',e=>{if(e.data&&e.data.type==='jobsWidgetResize'){document.getElementById('jobsWidget').style.height=e.data.height+'px'}});</script>
```
