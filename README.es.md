# Widget de Búsqueda de Empleo Local para Sitios Web

Widget de búsqueda de empleo ligero e integrable para sitios web, blogs, directorios y plataformas de medios.

Añade una experiencia de búsqueda de empleo local totalmente responsive a tu sitio web usando un único embed de iframe. El widget incluye búsqueda por palabras clave, autocompletado de ubicación, soporte de paginación y redimensionamiento automático de la altura del iframe.

Perfecto para:

* Portales de empleo
* Blogs
* Sitios de noticias locales
* Portales inmobiliarios
* Páginas de carreras
* Landing pages SEO
* Plataformas de publicación

# Instalación

Copia y pega este código en tu sitio web:

```html
<div class="jobs-widget-wrap"><iframe id="jobsWidget" src="https://jobs-in.us/display/jobs-search-widget.php" loading="lazy"></iframe><div class="jobs-widget-credit">Widget de búsqueda de empleo local by <a href="https://jobs-in.us/find/country,US" target="_blank">jobs-in.us</a></div></div><style>.jobs-widget-wrap{margin:40px 0}#jobsWidget{width:100%;height:760px;border:0;overflow:hidden;border-radius:24px;background:#f5f5f5;transition:height .25s ease}.jobs-widget-credit{margin-top:10px;text-align:right;font-size:13px;color:#777}.jobs-widget-credit a{color:#1a73e8;text-decoration:none}</style><script>window.addEventListener('message',e=>{if(e.data&&e.data.type==='jobsWidgetResize'){document.getElementById('jobsWidget').style.height=e.data.height+'px'}});</script>
```
