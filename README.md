# Local-Job-Search-Widget-for-Website

Lightweight embeddable jobs search widget for websites, blogs, directories and media platforms.

Add a fully responsive local jobs search experience to your website using a single iframe embed. The widget includes keyword search, location autocomplete, pagination support and automatic iframe height resizing.

Perfect for:

* Job boards
* Blogs
* Local news websites
* Real estate portals
* Career pages
* SEO landing pages
* Medium-style publishing platforms

---

# Features

* Responsive design
* Keyword autocomplete
* Location autocomplete
* Dynamic iframe auto height
* Pagination support
* Mobile friendly
* Fast lightweight embed
* Easy installation
* Cross-domain compatible
* SEO friendly integration

# Installation

Copy and paste this code into your website:

```html id="8d3d5h"
<div class="jobs-widget-wrap"><iframe id="jobsWidget" src="https://jobs-in.us/display/jobs-search-widget.php" loading="lazy"></iframe><div class="jobs-widget-credit">Local jobs search widget by <a href="https://jobs-in.us/find/country,US" target="_blank">jobs-in.us</a></div></div><style>.jobs-widget-wrap{margin:40px 0}#jobsWidget{width:100%;height:760px;border:0;overflow:hidden;border-radius:24px;background:#f5f5f5;transition:height .25s ease}.jobs-widget-credit{margin-top:10px;text-align:right;font-size:13px;color:#777}.jobs-widget-credit a{color:#1a73e8;text-decoration:none}</style><script>window.addEventListener('message',e=>{if(e.data&&e.data.type==='jobsWidgetResize'){document.getElementById('jobsWidget').style.height=e.data.height+'px'}});</script>
```
