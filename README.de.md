# Lokales Job-Such-Widget für Websites

Leichtgewichtiges einbettbares Stellensuch-Widget für Websites, Blogs, Verzeichnisse und Medienplattformen.

Fügen Sie Ihrer Website mit einem einzigen iframe-Einbettungscode ein vollständig responsives lokales Jobsuche-Erlebnis hinzu. Das Widget umfasst Stichwortsuche, Standort-Autovervollständigung, Paginierungsunterstützung und automatische iframe-Höhenanpassung.

Perfekt für:

* Jobbörsen
* Blogs
* Lokale Nachrichtenwebsites
* Immobilienportale
* Karriereseiten
* SEO-Landingpages
* Publishing-Plattformen

# Installation

Kopieren Sie diesen Code und fügen Sie ihn in Ihre Website ein:

```html
<div class="jobs-widget-wrap"><iframe id="jobsWidget" src="https://jobs-in.us/display/jobs-search-widget.php" loading="lazy"></iframe><div class="jobs-widget-credit">Lokales Job-Such-Widget von <a href="https://jobs-in.us/find/country,US" target="_blank">jobs-in.us</a></div></div><style>.jobs-widget-wrap{margin:40px 0}#jobsWidget{width:100%;height:760px;border:0;overflow:hidden;border-radius:24px;background:#f5f5f5;transition:height .25s ease}.jobs-widget-credit{margin-top:10px;text-align:right;font-size:13px;color:#777}.jobs-widget-credit a{color:#1a73e8;text-decoration:none}</style><script>window.addEventListener('message',e=>{if(e.data&&e.data.type==='jobsWidgetResize'){document.getElementById('jobsWidget').style.height=e.data.height+'px'}});</script>
```
