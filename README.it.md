# Widget di Ricerca Lavoro Locale per Siti Web

Widget di ricerca lavoro leggero e incorporabile per siti web, blog, directory e piattaforme media.

Aggiungi al tuo sito web un'esperienza di ricerca lavoro locale completamente responsive con un singolo embed iframe. Il widget include ricerca per parole chiave, completamento automatico della posizione, supporto alla paginazione e ridimensionamento automatico dell'altezza dell'iframe.

Perfetto per:

* Bacheche di lavoro
* Blog
* Siti di notizie locali
* Portali immobiliari
* Pagine carriera
* Landing page SEO
* Piattaforme di pubblicazione

# Installazione

Copia e incolla questo codice nel tuo sito web:

```html
<div class="jobs-widget-wrap"><iframe id="jobsWidget" src="https://jobs-in.us/display/jobs-search-widget.php" loading="lazy"></iframe><div class="jobs-widget-credit">Widget di ricerca lavoro locale by <a href="https://jobs-in.us/find/country,US" target="_blank">jobs-in.us</a></div></div><style>.jobs-widget-wrap{margin:40px 0}#jobsWidget{width:100%;height:760px;border:0;overflow:hidden;border-radius:24px;background:#f5f5f5;transition:height .25s ease}.jobs-widget-credit{margin-top:10px;text-align:right;font-size:13px;color:#777}.jobs-widget-credit a{color:#1a73e8;text-decoration:none}</style><script>window.addEventListener('message',e=>{if(e.data&&e.data.type==='jobsWidgetResize'){document.getElementById('jobsWidget').style.height=e.data.height+'px'}});</script>
```
