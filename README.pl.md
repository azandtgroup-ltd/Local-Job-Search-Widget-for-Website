# Lokalny Widget Wyszukiwania Pracy dla Stron Internetowych

Lekki, osadzalny widget wyszukiwania pracy dla stron internetowych, blogów, katalogów i platform medialnych.

Dodaj w pełni responsywne lokalne wyszukiwanie pracy do swojej strony za pomocą jednego osadzenia iframe. Widget zawiera wyszukiwanie słów kluczowych, autouzupełnianie lokalizacji, obsługę paginacji i automatyczne dopasowanie wysokości iframe.

Idealny dla:

* Portali pracy
* Blogów
* Lokalnych serwisów informacyjnych
* Portali nieruchomości
* Stron kariery
* Landing page SEO
* Platform publikacyjnych

---

# Funkcje

* Responsywny design
* Autouzupełnianie słów kluczowych
* Autouzupełnianie lokalizacji
* Dynamiczne automatyczne dopasowanie wysokości iframe
* Obsługa paginacji
* Przyjazny dla urządzeń mobilnych
* Szybkie lekkie osadzanie
* Łatwa instalacja
* Kompatybilność między domenami
* Integracja przyjazna dla SEO

# Instalacja

Skopiuj i wklej ten kod na swoją stronę:

```html
<div class="jobs-widget-wrap"><iframe id="jobsWidget" src="https://jobs-in.us/display/jobs-search-widget.php" loading="lazy"></iframe><div class="jobs-widget-credit">Lokalny widget wyszukiwania pracy od <a href="https://jobs-in.us/find/country,US" target="_blank">jobs-in.us</a></div></div><style>.jobs-widget-wrap{margin:40px 0}#jobsWidget{width:100%;height:760px;border:0;overflow:hidden;border-radius:24px;background:#f5f5f5;transition:height .25s ease}.jobs-widget-credit{margin-top:10px;text-align:right;font-size:13px;color:#777}.jobs-widget-credit a{color:#1a73e8;text-decoration:none}</style><script>window.addEventListener('message',e=>{if(e.data&&e.data.type==='jobsWidgetResize'){document.getElementById('jobsWidget').style.height=e.data.height+'px'}});</script>
```
