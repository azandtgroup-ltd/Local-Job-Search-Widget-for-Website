# Widget de Recherche d'Emploi Local pour Sites Web

Widget de recherche d'emploi léger et intégrable pour les sites web, blogs, annuaires et plateformes médias.

Ajoutez une expérience de recherche d'emploi locale entièrement responsive à votre site web grâce à un simple embed iframe. Le widget comprend la recherche par mots-clés, la saisie automatique de localisation, la pagination et le redimensionnement automatique de la hauteur de l'iframe.

Parfait pour :

* Les sites d'offres d'emploi
* Les blogs
* Les sites d'actualités locales
* Les portails immobiliers
* Les pages carrières
* Les pages de destination SEO
* Les plateformes de publication

---

# Installation

Copiez et collez ce code dans votre site web :

```html
<div class="jobs-widget-wrap"><iframe id="jobsWidget" src="https://jobs-in.us/display/jobs-search-widget.php" loading="lazy"></iframe><div class="jobs-widget-credit">Widget de recherche d'emploi local par <a href="https://jobs-in.us/find/country,US" target="_blank">jobs-in.us</a></div></div><style>.jobs-widget-wrap{margin:40px 0}#jobsWidget{width:100%;height:760px;border:0;overflow:hidden;border-radius:24px;background:#f5f5f5;transition:height .25s ease}.jobs-widget-credit{margin-top:10px;text-align:right;font-size:13px;color:#777}.jobs-widget-credit a{color:#1a73e8;text-decoration:none}</style><script>window.addEventListener('message',e=>{if(e.data&&e.data.type==='jobsWidgetResize'){document.getElementById('jobsWidget').style.height=e.data.height+'px'}});</script>
```
