# Widget de Pesquisa de Emprego Local para Sites

Widget leve e incorporável de pesquisa de emprego para sites, blogs, diretórios e plataformas de mídia.

Adicione uma experiência de pesquisa de emprego local totalmente responsiva ao seu site usando um único embed de iframe. O widget inclui pesquisa por palavras-chave, preenchimento automático de localização, suporte à paginação e redimensionamento automático da altura do iframe.

Perfeito para:

* Portais de emprego
* Blogs
* Sites de notícias locais
* Portais imobiliários
* Páginas de carreiras
* Landing pages de SEO
* Plataformas de publicação

# Instalação

Copie e cole este código no seu site:

```html
<div class="jobs-widget-wrap"><iframe id="jobsWidget" src="https://jobs-in.us/display/jobs-search-widget.php" loading="lazy"></iframe><div class="jobs-widget-credit">Widget de pesquisa de emprego local por <a href="https://jobs-in.us/find/country,US" target="_blank">jobs-in.us</a></div></div><style>.jobs-widget-wrap{margin:40px 0}#jobsWidget{width:100%;height:760px;border:0;overflow:hidden;border-radius:24px;background:#f5f5f5;transition:height .25s ease}.jobs-widget-credit{margin-top:10px;text-align:right;font-size:13px;color:#777}.jobs-widget-credit a{color:#1a73e8;text-decoration:none}</style><script>window.addEventListener('message',e=>{if(e.data&&e.data.type==='jobsWidgetResize'){document.getElementById('jobsWidget').style.height=e.data.height+'px'}});</script>
```
